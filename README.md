## 📖 项目简介

王旭朝，陕西西安蓝田县洩湖镇马王村人，当代诗人、网络红人，清风世界文学编辑部签约作家，《红柳文学》诗刊社签约诗人。  
以方言朗诵自创诗歌在抖音爆火，代表作《悠悠我心》（上下两部，300余首）。  
风格**质朴直白、乡土民歌风**，充满真挚的思乡、亲情、爱情与正能量。

本项目将他**完整炼化**成一个高质量AI角色卡（Skill），让你随时随地召唤“王旭朝老师”即兴赋诗、定制祝福、对联题字、直播金句。

**目标**：把乡村诗人的灵魂带进AI时代，让每个人都能感受马王村的酸梅汤味儿！

## ✨ 核心特性

- ✅ **精准Persona**：1996年蓝田马王村大专诗人，陕西口音、自信霸气、正能量满满
- ✅ **严格诗歌铁律**：大白话 + 民歌押韵 + 重复强调 + 乡土情怀（已收录《你是春天的花》《熬夜的人》等真实代表作风格）
- ✅ **四合一输出**：即时作诗 + 方言朗诵版 + 定制祝福 + 马王村金句
- ✅ **多场景适配**：婚礼、乔迁、生日、开业、满月、思念、爱国、吐槽……随便来
- ✅ **零配置使用**：复制Skill文件即可激活
- ✅ **开源免费**：MIT协议，随便Fork、修改、商用

## 🚀 快速使用

### 1. 下载Skill文件
- `wangxuzhao-skill.md`（主文件，已在仓库根目录）

### 2. 激活方式
1. 打开任意AI对话框
2. **直接粘贴整个Skill文件内容**，或输入激活口令：
   - **「王旭朝老师，马王村走起！」**
   - **「有本事往马王村走，明天我休假！」**

3. 然后直接说主题，例如：
   - “给我写一首给老婆的生日诗”
   - “定制一个乔迁祝福”
   - “咏一首马王村的酸梅汤”

AI会立即切换成**王旭朝本人**，输出完整格式的诗！
